## 2. Условие задачи (Вариант 9)

Пост диагностики автомобилей представляет собой **одноканальную СМО с отказами**.  
Заявка на диагностику, поступившая в момент, когда пост занят, получает отказ.

**Исходные данные:**

| Параметр | Обозначение | Значение |
|----------|-------------|---------|
| Интенсивность потока заявок | λ | 0,5 автомобиля в час |
| Средняя продолжительность диагностики | t_обс | 1,2 часа |

Все потоки событий в системе простейшие.

**Требуется:**  
Определить вероятностные характеристики СМО в установившемся режиме работы.

---

## 3. Теоретические основы

Для одноканальной СМО с отказами возможны два состояния системы:

- **S0** — канал свободен (ожидание заявки)
- **S1** — канал занят (идет обслуживание)

Переход из S0 в S1 происходит с интенсивностью λ (поступление заявки).  
Переход из S1 в S0 происходит с интенсивностью μ (окончание обслуживания).  
Заявка, поступившая в состоянии S1, получает отказ.

**Основные формулы:**

| Характеристика | Обозначение | Формула |
|----------------|-------------|--------|
| Интенсивность обслуживания | μ | μ = 1 / t_обс |
| Приведенная нагрузка | ρ | ρ = λ / μ |
| Вероятность простоя канала | P0 | P0 = 1 / (1 + ρ) |
| Вероятность отказа | P_отк | P_отк = ρ / (1 + ρ) |
| Относительная пропускная способность | q | q = 1 - P_отк |
| Абсолютная пропускная способность | A | A = λ × q |

---

## 4. Расчет характеристик

### 4.1. Интенсивность потока обслуживания

μ = 1 / t_обс = 1 / 1,2 = 0,8333 автомобиля/час

### 4.2. Приведенная интенсивность нагрузки

ρ = λ / μ = 0,5 / 0,8333 = 0,6

### 4.3. Вероятность простоя канала

P0 = 1 / (1 + ρ) = 1 / (1 + 0,6) = 1 / 1,6 = 0,625

**Интерпретация:** Пост диагностики простаивает 62,5% времени.

### 4.4. Вероятность отказа

P_отк = ρ / (1 + ρ) = 0,6 / 1,6 = 0,375

**Интерпретация:** 37,5% автомобилей получат отказ в обслуживании.

### 4.5. Относительная пропускная способность

q = 1 - P_отк = 1 - 0,375 = 0,625

### 4.6. Абсолютная пропускная способность

A = λ × q = 0,5 × 0,625 = 0,3125 автомобиля/час

## 6. Анализ и выводы

1. **Эффективность работы поста диагностики**  
   Относительная пропускная способность составляет 62,5%. Это означает, что из каждых 10 автомобилей, прибывающих на пост, в среднем 6–7 будут обслужены, а 3–4 получат отказ.

2. **Загрузка оборудования**  
   Канал занят в 37,5% времени, что свидетельствует о достаточной загрузке, но не о перегрузке системы.

3. **Практические рекомендации**  
   При данной интенсивности потока (0,5 авт/час) и времени обслуживания (1,2 часа) один пост диагностики справляется с нагрузкой, однако почти 40% клиентов теряются.  
   - Для снижения доли отказов до 5–10% необходимо либо увеличить количество постов (переход к многоканальной СМО), либо уменьшить время диагностики (повысить производительность).  
   - Альтернативно, можно организовать очередь (СМО с ожиданием), если позволяет площадь стоянки.

4. **Соответствие теории**  
   Полученные результаты соответствуют классической модели Эрланга для одноканальной СМО с отказами. Выполняется соотношение P0 + P_отк = 1, что подтверждает корректность расчетов.

## 7. Контрольные вопросы (ответы)

### 1. Что понимается под системой массового обслуживания?
**Ответ:** СМО — это система, в которую в случайные моменты времени поступают заявки на обслуживание, и которая располагает каналами для их обслуживания. Примеры: телефонная станция, пост диагностики, билетная касса.

### 2. Что является основными компонентами СМО?
**Ответ:** Входной поток заявок (интенсивность λ), каналы обслуживания (количество n, интенсивность μ), очередь (если есть), дисциплина обслуживания, выходной поток.

### 3. Что является предметом теории массового обслуживания?
**Ответ:** Построение математических моделей для анализа и расчета характеристик СМО: пропускной способности, вероятности отказа, времени ожидания, загрузки оборудования и др.

### 4. Какие виды СМО вы знаете?
**Ответ:**
- По числу каналов: одноканальные, многоканальные.
- По наличию очереди: с отказами, с ожиданием (ограниченной/неограниченной очередью).
- По дисциплине обслуживания: FIFO (первым пришел — первым обслужен), LIFO (последним пришел — первым обслужен), приоритетное обслуживание.

### 5. Перечислите основные функциональные характеристики одноканальной СМО с отказами.
**Ответ:**
- P0 — вероятность простоя канала;
- P_отк — вероятность отказа;
- q — относительная пропускная способность;
- A — абсолютная пропускная способность;
- ρ — приведенная нагрузка.
